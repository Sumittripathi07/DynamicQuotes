# DynamicQuotes

## Demo
https://dynamicquotes.netlify.app/



## Run Locally

Clone the project

```bash
  git clone https://github.com/Sumittripathi07/DynamicQuotes.git
```

Go to the project directory

```bash
  cd DynamicQuotes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
